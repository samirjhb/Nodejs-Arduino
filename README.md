

## Descripcion

Este proyecto tiene como finalidad la integracion entre Node y Arduino

## Intalacion

```bash
$ npm install
```

## Ejecutar Poyecto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
